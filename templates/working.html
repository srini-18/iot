How Our Project Works

Our Object Recognition project utilizes a webcam to capture real-time video frames. These frames are then processed using the YOLOv8 object detection model.

Working Steps:

  1. Webcam Input: The user's webcam provides the video stream.
  2. Frame Processing: Each video frame is analyzed by the YOLOv8 model.
  3. Object Detection: YOLOv8 identifies and classifies objects within the frame.
  4. Output Display: The detected objects are highlighted with bounding boxes and labels on the video feed.

This process happens continuously, providing a real-time object recognition experience.

Technical Details

We use Python, Flask, OpenCV, and the YOLOv8 library for this project.
