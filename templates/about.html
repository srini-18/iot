About This Project

This project demonstrates real-time object recognition using a webcam and YOLOv8.

YOLOv8 is a state-of-the-art object detection model that allows us to identify and classify multiple objects in a video stream.

Key Features

  • Real-time object detection
  • Webcam integration
  • Display of detected objects with confidence scores

Additional Information
